# My Winter Car VIN Decoder

The My Winter Car VIN Decoder reads a vehicle VIN either from the game's save file "carparts.txt" (created after the first save) or from a manually entered VIN, and decodes the following fields:

- COUNTRY
- ASSEMBLY PLANT
- MODEL
- BODY TYPE
- VERSION
- YEAR
- MONTH
- SERIAL NUMBER
- DRIVE
- ENGINE
- GEARBOX
- AXLE RATIO
- AXLE LOCK
- BODY COLOUR
- VINYL ROOF
- INTERIOR TRIM
- RADIO
- INSTRUMENT PANEL
- WINDSHIELD
- SEATS
- SUSPENSION
- BRAKES
- WHEELS
- REAR WINDOW
